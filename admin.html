Ajuda Rápida

Como funciona este painel?

Todas as configurações são salvas no localStorage do navegador, na chave bemorhSiteConfig. Isso significa que:

  • As alterações afetam imediatamente a página index.html neste mesmo navegador.
  • Se você abrir o site em outro navegador/computador, será necessário configurar novamente.
  • Para tornar isso definitivo para todos os visitantes, o ideal é futuramente salvar isso em um backend (ex: Firebase) e carregar os dados de lá.
Restaurar tudo ao padrão

Use o botão "Restaurar Padrão" na aba de Configurações. Isso irá:

  • Apagar a configuração salva em localStorage;
  • Recarregar todos os campos com os textos padrão que vieram no código original.
Problemas comuns
  • Textos não mudam no site: verifique se você salvou as alterações (botão "Salvar Configurações") e depois recarregue o index.html no navegador (F5).
  • Erros estranhos: clique em "Restaurar Padrão" e depois salve novamente.
